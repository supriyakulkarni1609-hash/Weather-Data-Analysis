# Task 1 – Weather Data Analysis

This repository contains my internship Task 1 submission.

## Overview
The given weather dataset was analyzed using Python. The analysis includes data exploration, visualization, and rainfall prediction using linear regression.

## Files Included
- weather.csv – Dataset provided for analysis
- weather_data_analysis.py – Python script for analysis

## Tools & Libraries
- Python
- pandas
- matplotlib
- seaborn
- scikit-learn

## How to Run
1. Keep both files in the same folder
2. Install dependencies:
   pip install pandas matplotlib seaborn scikit-learn
3. Run the script:
   python weather_data_analysis.py

## Output
- Dataset summary
- Graphical visualizations
- Rainfall prediction results
